# Vocab Bloom Hub — Frontend

Web interface for **Vocab Bloom Hub**, a modular open-source platform for working with vocabulary, linguistics data, and multilingual dictionaries.

Built with **Next.js**.

---

## 🚀 Overview

This frontend provides:

- 🔎 Word search and lookup
- 📖 Dictionary entry pages
- 🌐 Multilingual vocabulary browsing
- 🔗 Semantic relations visualization (synonyms, antonyms, hypernyms, etc.)
- 📚 Educational tools for language learning

It connects to the Vocab Bloom Hub backend API.

---

## 🧱 Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS (optional if you use it)
- REST API / (future: GraphQL or tRPC)
